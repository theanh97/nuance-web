# NUANCE DEMO LAUNCH SPECS (v1.0)

> **Context:** "Shock & Awe" Campaign for 2-Day Launch.
> **Goal:** Maximize "Feel" and "Atmosphere".

## 1. Feature: Spatial Audio (3D Sound)
**Problem:** Sound currently feels "flat" (centered).
**Solution:** Implement Stereophonic Panning based on pen position.
**Technical Specs:**
- **Module:** `src/core/SoundEngine.ts`
- **Component:** Add `PannerNode` to the audio graph.
- **Logic:**
    - Input: `panX` (Range: -1.0 Left to +1.0 Right).
    - Calculation: Map screen X coordinate `[0, windowWidth]` -> `[-1, 1]`.
    - Update Rate: On every `pointermove` event.

## 2. Feature: Visual Depth (Paper Texture)
**Problem:** White/Gray background feels "digital" and cheap.
**Solution:** High-fidelity paper texture with slight grain.
**Technical Specs:**
- **Asset:** `public/assets/textures/paper_grain.webp` (Generated by AI).
- **CSS:** Apply as `background-image` to `#root` or Canvas wrapper.
- **Blend Mode:** Use `multiply` blend mode for ink if possible, or just opaque ink on texture for MVP.

## 3. Feature: Precision Zoom (Pivot Fix)
**Problem:** Zooming scales from top-left (0,0) or center of screen, losing context.
**Solution:** Zoom towards the mouse pointer (Pivot Zoom).
**Technical Specs:**
- **Module:** `src/core/geminiInkRenderer.ts`
- **Math:**
    ```typescript
    // World = (Screen - Pan) / Zoom
    // NewPan = Screen - (World * NewZoom)
    ```
- **Constraint:** Keep Pan/Zoom logic cleanly separated in `requestRedraw`.

## System Integration
- All features must be toggleable via `config` object in future, but hardcoded ON for Demo.
